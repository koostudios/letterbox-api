# letterbox API

A lightweight RESTful backend for supercharging online forms.
  
**Latest Version:** 0.0.1

## Features

- **Easy setup:** Generates code for your form and then starts collecting straight away!
- **Simple but flexible:** As long as it is in a form format, letterbox will collect it.
- **Powered by the Cloud:** Built on trusted cloud technologies like Node.js and Cloudant to keep your data safe
